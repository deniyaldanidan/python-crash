## Logging Module Notes:

### Levels:

| Level    | Value | Method               |
| -------- | ----- | -------------------- |
| DEBUG    | 10    | `logging.debug()`    |
| INFO     | 20    | `logging.info()`     |
| WARNING  | 30    | `logging.warning()`  |
| Error    | 40    | `logging.error()`    |
| Critical | 50    | `logging.critical()` |
---------
